# types of addresses 
1. an ip address (changeable)
2. a mac address (unchangeable)
## ip address 
- internet protocol address is the address which can be assigned to a network for a short period of time and then can be assigned to another device 
- ex 192.168.1.1 
- the numbers are calculated through ip addressing or subnetting 
- to share data amongst devices we can use a private ip
- to share data on the internet we require a public ip 
- ipv4 has around 4.29 billion addreses 
- ipv6 has around 340 trillion plus it also consists of alphabets
## mac address
- all devices have a nic each nic is equipped with its own unique mac address
- a mac address is of twelve characters it is hexadecimal
- the first 6 characters represent the vendors number while the later 6 represent a unique number
>spoofing is the process of faking a mac address on the internet 
- we can access a public wifi which is usually paid for by acting as someone who has already paid for the wifi and impersonating that persons mac address
# ping 
- it is one of the most important networking tools available to us
- it uses ICMP (internet control message protocol) packets to determine the performance of a connection 
- it measures the time taken for ICMP packets travelling between devices done using ICMP echo packet and echo reply 
- syntax is ping ip address or ping 0.0.0.0 or whatever the address is
